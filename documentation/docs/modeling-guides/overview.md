# Modeling Guides

A **Modeling Guide** is a guided walkthrough that takes a single example domain from an empty repository to a coherent ESDM model. Each guide has its own example, its own narrative, and its own rhythm; later chapters build on earlier ones, and you can read a guide top to bottom.

This is the chapter for hands-on, narrative learning. **[Concepts](/concepts/overview.md)** explains what each piece of the model means in isolation; the Guides show how the pieces fit together when you actually build something. **[Recipes](/recipes/overview.md)** answers single-question problems; the Guides take a longer view.

## Available Guides

The first Modeling Guide is in preparation. When it lands, it will appear here with a short description, a link to its first chapter, and an estimate of how long the walkthrough takes end-to-end.

If you want to dive in before the Guides are ready, **[Getting Started](/getting-started/installing-esdm.md)** is the place to begin – it covers installation, your first model, and the most common CLI invocations.
