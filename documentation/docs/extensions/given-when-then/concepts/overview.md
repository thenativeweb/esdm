# Concepts

The Given-When-Then extension defines two structures, each on its own page:

- A **[Feature](/extensions/given-when-then/concepts/feature.md)** is the top-level document, grouping related Scenarios about one consistency unit.
- A **[Scenario](/extensions/given-when-then/concepts/scenario.md)** is a nested entry inside a Feature: one behavioral assertion in the canonical Given/When/Then form.
